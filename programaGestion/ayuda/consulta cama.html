Consulta Cama

  • Buscar

    • Sirve para buscar una cadena.

  • Campo de consulta

    • muestra las camas registradas. Si has introducido una cadena en el campo de búsqueda muestra las camas registradas que contengan la cadena introducida en el campo de búsqueda.

  • consultar

    El botón consultar realiza una consulta de la cama seleccionada en la base de datos.

      Consultar cama

    • Número Cama

      • Muestra el número de la cama seleccionada

    • ID Paciente

      • Muestra el ID del paciente que ocupa la cama seleccionada.

    • ID Edificio

      • Muestra el ID del edificio donde se encuentra la cama seleccionada.

    • Aceptar

      • El botón aceptar cierra la consulta de la cama.
